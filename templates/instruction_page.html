  • User can find all the projects in which they are tagged with My Projects filter button.
  • A user can change the status of the tickets assigned for all the tagged projects.
  • User can delete the notifications. The color of all the seen notifications will be white where as it will different for unseen notifications.
  • User can see all the notifications by clicking on the What's New button in dashboard.
  • User can filter projects category wise and project manager-wise from the Project Table.
  • User can navigate to specific project detail from the project table.
  • User can find all the assigned tickets from the My Tickets link
  • On the My Ticket page user can filter all the tickets all the possible combination of the filtering parameters
  • Filtering parameters are project, task status, task category, new task, old task, nearing deadline task
  • Ticket's color will depend upon the status of the ticket such as open, in progress, complete, backlog etc
  • Ticket's color will change automatically if the deadline is ended and the task status is not complete
  • User can change the status of task by clicking on it, However the user can not change the status of a task which is complete
  • Any user can be a Project Manager. However, only a django admin user can assign role of Project Manager to a user.
  • A Project can have only one project Manager.
  • A project manager can create new project from nav-bar which will be visible only to Project Manager.
  • A project manager can edit and delete a project from Project table.
  • Edit Project Page will include a search bar which will allow the Project Manager to search users and add or remove them from the project
  • User can also edit the project from the edit project page, also project manager can navigate to the manage ticket and create ticket page.
  • Manage ticket page gives a Project Manager full control over management of all the ticket for which user is Project Manager
  • Project Manager can filter all the tickets with different combinations of the filter parameters
  • Project Manager can delete or edit a ticket any ticket
  • If a project manager updates a ticket either by extending the deadline or by changing the category its status will be updagted to Open
  • A ticket once created with a specific project can not be tagged to another project it can then only be deleted.
